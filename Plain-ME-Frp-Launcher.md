项目介绍
----

Plain ME Frp Launcher X (PML 2) 是由 **RYCB Studio** 使用 .NET 10.0 开发的 ME-Frp 第三方客户端。 该客户端是对 ME Frp (幻缘映射) 的图形化实现，提供了简单便捷的操作，可以快速启动隧道，支持 Windows, Linux 和 MacOS 三端。 _也是目前三个产品中唯一一个支持跨平台的软件。_

### 主要特性

* 作者：**RYCB Studio**
* 命名灵感：**PCL Ⅱ**
* 基于 .NET 10.0 框架，跨平台支持
* 支持 Windows、Linux、MacOS 三端
* 操作简单便捷，功能丰富
* 持续更新，社区活跃

### 系统要求

* Windows 10/11/Server 2019+
* 常见 Linux x64 发行版
* macOS（Intel/Apple Silicon）

使用方法
----

#### 教程：如何在 Linux 上顺利运行本软件？

1. 1. 打开终端
2. 2. 输入 `bash <(curl -sSL https://content.rycb.mxj.pub/files/mefl/install.sh)`
3. 3. 等待安装完成
4. 4. 之后可直接输入 (2.1.0.0 以下) `mefrplauncherx` 或 (2.1.0.0 及以上) `pml-2` 启动

#### 为什么建议在 Linux 中使用终端运行？

Linux 软件安装目录通常不在用户主目录下，无法双击运行。使用终端可追踪程序运行情况，便于错误排查。

快速修复
----

### Windows - 软件无法启动

**问题重现：**点击软件，窗口闪退 / 无响应 / 白色边框。

**原因：**特定目录下需要管理员权限。

**解决方法：**

1. 1. 打开安装目录（如 `C:\Program Files\...`）
2. 2. 右键 `MEFrpLauncherX.exe` → 属性
3. 3. 切换到"兼容性" → 点击"更改所有用户的设置"
4. 4. 勾选"以管理员身份运行此程序" → 确定
5. 5. 重新启动软件

### Linux - 软件无法启动

**问题重现：**无窗口，控制台报错 "UnauthorizedException"。

**原因：**权限不足。

**解决方法：**

1. 1. (2.1.0.0 以下)打开终端，执行：`chmod -R a+r,a+w /usr/share/mefrplauncherx/*`
2. 2. (2.1.0.0 及以上)打开终端，执行：`chmod -R a+r,a+w /opt/pml-2`
3. 3. 重新运行：(2.1.0.0 以下) `mefrplauncherx` 或(2.1.0.0 及以上) `pml-2`
