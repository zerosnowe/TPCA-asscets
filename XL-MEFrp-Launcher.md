## 项目介绍

XL-ME-Frp-Launcher 是由 yealqp 使用 Tauri 2 框架开发的 ME-Frp 第三方客户端。
界面高仿官网样式，可能是目前收录的三个客户端中最美观的一个，也可能也是包体最小的一个，亦或是 bug 最少的一个。

### 主要特性

- 基于 Tauri 2 框架，性能优异
- 界面高仿官网样式，美观易用
- 提供官网没有的启停隧道功能
- 拥有开机自启动、系统托盘等功能
- 包体小巧，启动迅速
- 支持 Token 登录功能

### 系统要求

- Windows 10/11
- Windows Server 2019/2022/2025
- 需要安装 WebView2 运行时

## 下载安装

[OpenList 下载](https://alist.yealqp.cn/ME-Frp%20XL%20%E5%AE%A2%E6%88%B7%E7%AB%AF)

#### 系统要求

本软件仅适用于 Windows 10, Windows 11, Windows Server 2019, 2022, 2025。在使用前请确认您的电脑上安装了 WebView2（WebView2 已经在 Windows 10 1803 以上和更高版本的 Windows 上默认安装）。

#### 使用说明

请使用 ME-Frp 官网账号登录。需要更多功能或报告 Bug 请向邮箱 yealqp@163.com 发送邮件。

### 安装步骤

1. 从官方链接下载适合您操作系统的安装包
2. 运行安装程序，按照提示完成安装
3. 启动程序，输入您的 ME-Frp 账户信息
